## plugin-zawgyiprocss
it's origin form Saturngod Plugin Zawgyi CSS
To support smaart zawgyi pro font family and font embed in gitbook web version.


### To Install

```
npm install plugin-zawgyiprocss
```

In `book.json` ,

add `plugin-zawgyiprocss` in `plugins`

```
"plugins": [
      "plugin-zawgyiprocss"
      ]
```
